# ADR-008: Lite/Company Dual Mode

- 상태: accepted
- 날짜: 2026-03-06
- 관련: ADR-007 (플러그인 시스템)

## 맥락

7팀 컴퍼니 모델은 Jarvis의 핵심 차별점이지만 진입장벽이 높음.
새 사용자가 5분 내 봇을 가동하려면 단순한 시작점 필요.

## 결정

`jarvis init` 위저드에서 두 가지 모드 선택 가능:

| 모드 | 내용 | 대상 |
|------|------|------|
| **Lite** | 봇 + 3개 기본 태스크 (standup, summary, health) | 첫 사용자 |
| **Company** | 7팀 + context-bus + shared-inbox + ADR | 파워유저 |

### 온보딩 위저드 (`jarvis-init.sh`)
5가지 질문으로 전체 설정 자동 생성:
1. 이름 / 타임존
2. 모드 (Lite / Company)
3. Discord 봇 토큰
4. LLM 프로바이더 (Claude Max / API / Ollama)
5. Obsidian Vault 경로 (선택)

### 모드 전환
- Lite → Company: `jarvis-init.sh` 재실행, Company 선택
- 설정 상태: `~/.jarvis/.initialized` 파일에 저장

## 대안

- 단일 모드만 제공: 진입장벽 때문에 채택률 하락 예상.
- 자동 감지: 사용자 의도를 추정하기 어려움.

## 결과

- 첫 사용자: 5분 내 기본 봇 가동
- 파워유저: 점진적으로 팀 추가, 커스텀 플러그인 설치
- 오픈소스 온보딩 장벽 대폭 감소
